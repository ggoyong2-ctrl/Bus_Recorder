# 🚌 서울시내버스 운행기록 프로그램 (Seoul_Bus_Drive_Recorder)

> **<ins>주의 : 이 프로그램을 사용하기 위해서는 반드시 공공데이터포털 OPEN API 사용신청을 해야합니다.</ins>**

## 🐍 파이썬(Python) 다운로드 및 설치 방법

이 프로그램을 실행하려면 컴퓨터에 파이썬이 설치되어 있어야 합니다.

**1.** 파이썬 공식 홈페이지[(https://www.python.org/downloads/)](https://www.python.org/downloads/)에 접속합니다.<br>
**2.** 화면 중앙의 노란색 **Download Python 3.x.x** (최신 버전) 버튼을 클릭하여 설치 파일을 다운로드합니다.<br>
**3.** 다운로드된 설치 파일을 더블클릭하여 실행합니다.

> ⚠️ **매우 중요 (설치 시 주의사항)**
> 설치 프로그램 첫 화면 하단에 있는 **`Add Python 3.x to PATH`** 체크박스를 **반드시 체크**한 후 `Install Now`를 클릭해 주세요! (이것을 체크하지 않으면 터미널에서 파이썬을 인식하지 못합니다.)

<br>

## 📦 필수 라이브러리 설치 방법

프로그램이 공공데이터를 불러오고 엑셀 파일로 저장하기 위해 필요한 추가 부품(라이브러리)들을 설치하는 과정입니다. 

**1.** 명령 프롬프트(Windows는 `cmd` 검색 후 실행, macOS는 `터미널` 실행)를 엽니다.<br>
**2.** 아래 명령어를 복사하여 붙여넣고 `Enter` 키를 누르면 자동으로 설치가 진행됩니다.

**📌 필수 라이브러리 (반드시 설치해야 합니다)**
```bash
pip install requests pandas openpyxl
```

### 🎨 선택 라이브러리 (설치를 권장합니다)

> 프로그램 내 이미지 해상도를 높여주고, 입력하신 인증키를 안전하게 암호화하여 저장하는 데 도움을 줍니다.

```bash
pip install Pillow cryptography
```
> 💡 **설치 팁**
> 특히 파이썬 설치할 때 `Add to PATH`를 체크하는 것은 처음 접하시는 분들이 가장 많이 실수하시는 부분이라 강조 표시(⚠️)를 넣어두었습니다. 사용하시면서 추가로 더 보완하고 싶은 내용이나 궁금한 점이 있으시면 언제든 편하게 말씀해 주세요!

## 📖 사용 방법

**1.** 🔑 공공데이터포털에서 발급받은 **인증키를 프로그램에 입력**합니다.<br>
**2.** 🚌 기록하고자 하는 **노선 검색 및 노선 설정**을 진행합니다.<br>
**3.** ▶️ **[기록 시작]** 버튼을 클릭합니다.<br>
**4.** 📡 해당 노선의 운행 출발과 운행 종료 데이터를 프로그램이 **자동으로 수집**합니다.<br>
**5.** 📊 실행 파일이 위치한 경로에 저장된 **엑셀 파일을 확인 및 사용**합니다.<br>

## 🔑 인증키 발급 및 Open API 사용신청 방법

**1.** 공공데이터포털 (https://www.data.go.kr) 에서 회원가입을 진행해주세요.

**2.** 아래 페이지에 각각 접속하여 API 활용신청을 하면, 인증키가 발급됩니다.
* **인증키 확인방법:** 로그인 ➔ 마이페이지 ➔ 개인 API인증키
* ⚠️ **참고:** 발급된 인증키는 보통 매주 월요일에 사용 승인됩니다. (사용 승인되기 전에는 발급된 인증키를 입력하더라도 사용 불가)

  * 📌 서울특별시_버스도착정보조회 서비스: https://www.data.go.kr/data/15000314/openapi.do
  * 📌 서울특별시_정류소정보조회 서비스: https://www.data.go.kr/data/15000303/openapi.do
  * 📌 서울특별시_버스위치정보조회 서비스: https://www.data.go.kr/data/15000332/openapi.do
  * 📌 서울특별시_노선정보조회 서비스: https://www.data.go.kr/data/15000193/openapi.do

**3.** 각각의 API 호출명 당 일일 **1,000 트래픽**이 부여됩니다.
* 호출명 당 1,000건의 제한은 이 프로그램을 사용하는데 다소 부족할 수 있습니다. (데이터 갱신 주기를 80~90초 정도로 느리게 설정하면 1,000건 이내에서 1개 노선의 하루 데이터를 받을 수 있으나, 실제 운행과 데이터 간 오차가 커지거나 일부 누락 가능성이 존재합니다.)

**4.** 오픈소스로 공개해드린 파이썬 코드를 참고하여 자신만의 프로그램을 새롭게 작성하신 경우, 본인의 프로그램을 활용사례로 신청하시면 각각의 API 호출명 당 일일 트래픽이 **10,000건**으로 상향됩니다.
* 10,000건이면 여러 노선을 하루종일 촘촘하게, 적은 오차값으로 모니터링 가능합니다.

**5.** API 호출 트래픽이 부족할 때, **트래픽 상향을 위한 활용사례 신청 방법**
> 공공데이터포털 (https://www.data.go.kr) 로그인 ➔ 마이페이지 ➔ 데이터 활용 ➔ Open API ➔ 활용신청 현황 ➔ 각각 활용신청 했던 서비스 클릭 ➔ **활용신청 버튼** ➔ 본인의 활용 사례 입력 (개발한 프로그램 스크린샷, 사이트 주소 등)
